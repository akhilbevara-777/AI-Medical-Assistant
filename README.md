# AI Medical Assistant 🩺

An AI-powered Medical Assistant built using **HTML, JavaScript, n8n, and Ollama AI**.

This project allows users to enter symptoms and receive:
- Disease prediction
- Explanation
- Medicine suggestions
- Health suggestions

---

# 🚀 Features

- Simple medical symptom checker
- AI-generated disease analysis
- Medicine recommendations
- Suggestions for recovery
- n8n workflow automation
- Ollama local AI integration
- Beginner-friendly project

---

# 🛠 Technologies Used

- HTML
- CSS
- JavaScript
- n8n
- Ollama
- AI Agent Workflow
